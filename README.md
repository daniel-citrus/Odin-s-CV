## About The Project

Odin's CV is a mobile-first application that allows users to generate pre-formatted resume documents that they can print out. Users can select from a variety of resume templates and fill out a form with their information, which will be used to generate a professional resume.

The application consists of two main parts:
1. **Input Form**: A comprehensive form for entering personal and professional information
2. **Output Window**: A live preview of the resume as users complete the form, with print functionality

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username/Odin's-CV.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```

## Usage

The application features a comprehensive form with the following sections:

- **Template Selection**: Choose from various resume templates
- **General Information**: Name, email, phone, and links
- **Educational Experience**: Multiple institutions with dates, location, degree, GPA, and relevant courses
- **Experience Section**: Company details, role, dates, location, and accomplishments
- **Projects**: Project name, technologies used, and descriptions
- **Technical Skills**: Languages and technologies
- **Certificates**: Certificate name, award year, and description

The output window provides a live preview formatted according to the selected template with a print button.
